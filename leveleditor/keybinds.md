# Keybinds

* **Up/Down/Left/Right** - Move cursor
* **O** - Open File
* **S** - Save File
* **Spacebar** - Cycle current block
* **F** - Place floor
* **L** - Line Tool
* **Delete/Backspace** - Delete current block
* **Escape** - Quit

## Block types (in order of cycle)

* "block"
* "platform"
* "coin"
* "spikes"
* "invisible_block"
* "finish_block"

## More info

To delete floors, first hover over the top left corner.
